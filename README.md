# ONLINE MARKETING CAMPAIGNE DASHBOARD IN POWER BI

![Project Banner](./Images/logo.avif)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This repository contains the **ONLINE MARKETING CAMPAIGNE DASHBOARD** created in Power BI. It includes detailed information about online campaigns on various social platforms during one month. The dashboard has many marketing metrics like Return on Marketing Investment, Contribution Margin, Click Through Rate, Lead Rate, Order Rate, Conversion Rate, etc.

The project was an attempt to explore Power BI‚Äôs capabilities for creating app-like dashboards with modern design visuals.

## Objectives

The project aims to answer questions such as:
- What are the general Marketing Spends and Revenue?
- What Marketing Campaigns had the best results and what the worst?
- What are the sum of expences on marketing by Instruments?
- What Digital Instruments (Social Media Platforms) makes the best Return on Marketing Investment?
- What Instrument has the lowest cost per oder?
- What are convertion rates?
- How many orders generated each instrument?

## Dataset

**Size:** 308 rows

| Column          | Description                                             |
|-----------------|---------------------------------------------------------|
| id              | Unique identifier for each record                       |
| c_date          | Date of the campaign data                                |
| campaign_name   | Name of the marketing campaign                           |
| category        | Category or type of the campaign                         |
| campaign_id     | Unique identifier for the campaign                       |
| impressions     | Number of times the campaign was displayed               |
| mark_spent      | Amount of money spent on the campaign                    |
| clicks          | Number of clicks generated by the campaign               |
| leads           | Number of leads captured from the campaign               |
| orders          | Number of orders resulting from the campaign             |
| revenue         | Total revenue generated from the campaign                |

## App Structure

1. **GENERAL MARKETING INFO** ‚Äì Info about marketing expences and revenue, contribution margin by instruments
2. **FINANCIAL PERFORMANCE BY MARKETING INSTRUMENT** ‚Äì Return on marketing investment by instruments, impressions, clicks, leads, orders
3. **ACQUISITION FUNNEL BY MARKETING INSTRUMENT** ‚Äì Count of orders by instruments, click through rate, lead Rate, order Rate, conversion Rate, cost per order 
4. **CONCLUSION NOTES** ‚Äì draft insights from analysis

## Requirements

To open the app:

1. Install **Power BI Desktop** (version 2.149.1429.0 recommended) from the [official website](https://www.microsoft.com/en-us/download/details.aspx?id=58494).  
2. Open the file: `MarketingProject.pbix`  
3. To update the dataset, reconnect the `.csv` files from the Dataset folder using **Power Query**.
